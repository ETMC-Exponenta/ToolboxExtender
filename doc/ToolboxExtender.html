ToolboxExtender (class)

Control your toolbox. Core class for ToolboxDev, ToolboxStorage and ToolboxUpdater
Table of Contents
How to add Properties Methods Examples

How to add

To add Extender class to current project folder execute
ToolboxExtender.add('extender')

Properties

name - name of toolbox, char
pname - name of toolbox project file, char
type - type of toolbox ('toolbox'/'app'), char
remote - remote address (GitHub), char
vc - current installed toolbox version, char
root - project root folder, char
extv - ToolboxExtender version added to toolbox, char

Methods

TE = ToolboxExtender([root])
Create TE object of ToolboxExtender class. Optionally: you can specify custom toolbox folder (root). The default root is folder where class is stored.
install([fpath])
Install toolbox from current project .mltbx-file or from file specified by fpath (optional)
uninstall()
Uninstall current project toolbox if installed
vc = gvc()
Get current installed version
doc()
Open GettingStarted from doc folder
doc([name])
Open GettingStarted.html or another documentation file from doc folder specified by name (optional):
  • without extension: file 'name.html' will be opened
  • with .mlx extension: file 'name.mlx' will be opened
examples()
Change current directory to examples
web()
Open GitHub page of Toolbox
addfav(label, code, [icon])
Create MATLAB favorite with label and icon (optional) that runs specified MATLAB code. Toolbar shortcut will be created too. Favorite is placed in favorites category created for toolbox.
isfav(label)
Check does favorite label exist.
rmfav(label)
Remove favorite label from favorites and toolbar.
isfavs()
Check does favorites category created for toolbox exist.
rmfavs()
Remove favorites category created for toolbox.
