ToolboxExtender (class)

Control your toolbox. Core class for ToolboxDev, ToolboxStorage and ToolboxUpdater

Properties

name - name of toolbox
pname - name of toolbox project file
type - type of toolbox ('toolbox'/'app')
remote - remote address (GitHub)
pv - toolbox project version (from project file)
cv - current installed toolbox version
rv - remote toolbox version from GitHub
root - project root folder
extv - ToolboxExtender version initialized in project folder

Methods

init([classname])
Initialize in the curent toolbox project folder a copy of the ToolboxExtender class and classname (optional):
  • 'all' - ToolboxExtender, ToolboxDev, ToolboxStorage, ToolboxUpdater
  • 'extender' or without argument - ToolboxExtender
  • 'dev' - ToolboxExtender, ToolboxDev
  • 'storage' - ToolboxExtender, ToolboxStorage
  • 'updater' - ToolboxExtender, ToolboxUpdater
Initialized classes will have names depended on the project name, i.e.: ProjectNameExtender, ProjectNameDev, etc.
install([fpath])
Install toolbox from current project .mltbx-file or from file specified by fpath (optional)
installweb()
Install toolbox from remote (GitHub)
uninstall()
Uninstall current project toolbox if installed
pv = gpv()
Get project version
cv = gcv()
Get current installed version
rv = grv()
Get remote (GitHub) version
doc()
Open GettingStarted from doc folder
doc([name])
Open GettingStarted.html or another documentation file from doc folder specified by name (optional):
  • without extension: file 'name.html' will be opened
  • with .mlx extension: file 'name.mlx' will be opened
examples()
Change current directory to examples
