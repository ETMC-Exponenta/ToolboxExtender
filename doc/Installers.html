How to add

To add install.m and instalweb.m scripts class to current project folder execute
ToolboxExtender.add('install')

Web installer: installweb.m

installweb.m script downloads the latest version of .mltbx file from GitHub and installs it as a toolbox.

Why to use

This is the easiest way for users to install the latest version of your toolbox directly from the web via the only one command.

How to use

First of all create release of new version of your toolbox on GitHub.
Then add install scripts to your project:
ToolboxExtender.add('install')
After it you will see in command window 'Web installer command', i.e. for Toolbox Extender:
eval(webread('https://git.io/fjo2h'))
The URL leads to installweb.m script on GitHub.
Share this command with users via README.md or another way. When user executes this command he will have toolbox installed.

Use case

Provide your users with one-line MATLAB command that installs your toolbox directly form GitHub to MATLAB or MATLAB Online.

Local installer: install.m

install.m script builds .mltbx file from current folder and installs it as a toolbox.

Why to use

It's a bad idea to commit a .mltbx binary file to GitHub among projeсt files because it's a generated not source file.
Instead of .mltbx you can commit install.m and users who download your project as an archive will run this script and get toolbox installed.

How to use

Execute from current project folder
install

Use case

If user has downloaded an archive of your toolbox project files from GitHub, he unpacks the archive, goes to the project folder and executes install to get toolbox compilated and installed.
