ToolboxUpdater (class)

Add to your toolbox ability to update itself from remote (GitHub)

Properties

TE - ToolboxExtender object
vr - latest remote version from internet (GitHub)

Methods

TU = ToolboxUpdater([extender])
Create TU object of ToolboxStorage class.
If you call ToolboxUpdater class with optional ToolboxExtender object extender, TU will use this object, otherwise new ToolboxExtender object will be created inside TU (TE property)
ison = isonline()
Check internet is available
vr = gvr()
Get latest remote (GitHub) version vr
isupd = isupdate()
Check update is available
[vc, vr] = ver()
Display and get current (vc) and remote (vr) versions. If update is available show an update hint
update()
Install latest version from remote (GitHub) if update is available
installweb()
Install toolbox from remote (GitHub)

Additional information

How to integrate

Firstly you need to create a release of your toolbox on GitHub with installation binary atached. You can use ToolboxDev to automate this process.
After it add to yout toolbox function (or method to your app) that automatically checks that a newer version of toolbox are available, informs a user about it and allows him to install the latest version.
