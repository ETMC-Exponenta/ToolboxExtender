ToolboxStorage (class)

Add to your toolbox ability to store data (i.e. settings) in installation folder

Properties

TE - ToolboxExtender object
root - data folder (Storage root)
fname - name of .mat-file where data is stored (Storage file)
data - data that stored in Storage
auto - automatically save data to Storage when it changes (default: true)

Methods

TS = ToolboxStorage([fname],['TE',extender],['local',true])
Create TS object of ToolboxStorage class. Optionally: you can specify custom data file name fname.
Data file will be created in ...\MATLAB Add-Ons\Data folder. So you can reinstall or delete toolbox and data will not be deleted.
If you call ToolboxStorage class with additional parameter 'TE' with value of ToolboxExtender object extender, TS will use this object, otherwise new ToolboxExtender object will be created inside TS (TE property)
If you call ToolboxStorage class with additional parameter 'local' with value true, data file will be created in the toolbox root folder.
data = load([fpath])
Load data from Storage file or from file specified by fpath (optional)
save([data], [fpath])
Save Storage data to Storage file. Optional: save data, optional: to fpath
Check update is available
data = set(varname, value, [type])
Set value of varname variable in Storage data. Optionally you can specify type of variable
[value, isf] = get(varname, [type])
Get value of varname variable from Storage data. Optionally you can specify type of variable. isf tells you either varname exists in data or doesn't
[fpath, fname] = getpath()
Get path to Storage data file (fpath) and name of this file (fname)
open()
Open Storage data file in system file manager
data = import(fpath)
Import data from other Storage data file (specified by fpath) to current Storage
export(fpath)
Export Storage data to file specified by fpath
clear()
Clear Storage data and delete data file

Examples

Create an object of the ToolboxStorage
TS = ToolboxStorage
TS =
ToolboxStorage with properties: E: [1×1 ToolboxExtender] root: 'D:\MATLAB\ToolboxExtender' fname: "ToolboxExtender_data" data: [1×1 struct] auto: 1
Check data file path
fpath = TS.getpath
fpath = "D:\MATLAB\ToolboxExtender\ToolboxExtender_data.mat"
Add new field to data (data will be saved automatically to data file)
TS.set('testvar', 1:3)
ans = struct with fields:
testvar: [1 2 3]
Get saved field from data
testvar = TS.get('testvar')
testvar = 1×3
1 2 3
Clear storage and delete data file
TS.clear
